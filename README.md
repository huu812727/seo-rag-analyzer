# 🚀 RAG-система для SEO-анализа конкурентов (ТОП-10)

Профессиональный инструмент автоматизации глубокого SEO-анализа на базе технологий Retrieval-Augmented Generation (RAG). Система автономно исследует поисковую выдачу, парсит контент лидеров и генерирует структурированный экспертный отчет уровня Senior SEO.

---

### 💼 Бизнес-ценность
Инструмент разработан для кратного ускорения процессов в SEO-агентствах и инхаус-командах:
- **Экономия времени:** Сокращает время на ручной сбор и анализ данных конкурентов с **2-3 часов до 3-5 минут**.
- **Экономическая эффективность:** Благодаря использованию модели **Gemini 3 Flash**, стоимость глубокого анализа одного кластера составляет **менее $0.01**, что в десятки раз дешевле ручного труда или использования проприетарных SEO-платформ.
- **Масштабируемость:** Позволяет автоматизировать первичный аудит и сбор семантических сущностей (LSI/Entity) для десятков проектов одновременно.

---

### 🏗 Архитектура (Pipeline)
Система представляет собой модульный конвейер, реализованный на современном стеке ИИ-технологий:

1.  **Frontend (UI):** `Streamlit` — интуитивный веб-интерфейс для ввода запросов и визуализации прогресса.
2.  **Search Engine:** `SerpApi (Google Search)` — динамическое получение актуального ТОП-10 органической выдачи.
3.  **Data Extraction:** `Firecrawl` — "умный" парсинг сайтов с автоматической конвертацией в Markdown и глубокой очисткой от технического мусора (JS, CSS, реклама).
4.  **Vector Store & Search:** 
    - `Sentence Transformers (all-MiniLM-L6-v2)` — генерация эмбеддингов.
    - `Pinecone` — высокопроизводительная векторная БД для хранения и поиска релевантных фрагментов контента.
5.  **LLM Orchestration:** `LangChain` + `Gemini 3 Flash` (via OpenRouter) — генерация экспертного англоязычного отчета и последующая профессиональная локализация на русский язык.

---

### ⚡ Киллер-фичи (Killer Features)
- **🛡 Защита от GIGO (Garbage In, Garbage Out):** Встроенная валидация данных. Система автоматически отфильтровывает битые ссылки, 404 страницы, заглушки Cloudflare/Captcha и слишком короткие тексты ( < 1000 символов).
- **🔄 Разделенная генерация и перевод:** Архитектура разделяет процесс логического анализа и лингвистической локализации. Это повышает стабильность и гарантирует сохранение профессиональной терминологии (LSI, H1, E-E-A-T).
- **🎯 Расширенный контекст:** Функция `retriever` настроена на извлечение до 40 наиболее релевантных чанков контента из всех ТОП-10 статей, что дает нейросети максимально полную картину.

### 🛠 Установка и запуск

1. **Клонирование и среда:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Настройка переменных окружения:**
   Создайте файл `.env` в корне проекта (по шаблону `.env_template`) и укажите ключи:
   - `SERPAPI_API_KEY`
   - `FIRECRAWL_API_KEY`
   - `PINECONE_API_KEY`
   - `OPENROUTER_API_KEY`

3. **Запуск приложения:**
   ```bash
   streamlit run app.py
   ```

### 📝 Структура формируемого отчета
- **Executive Summary:** Почему конкуренты в ТОПе (технический вес, паттерны).
- **Content Skeleton:** Идеальный H1 и структура H2-H3 на основе частотного анализа.
- **Semantic Entity Map:** Карта тематических сущностей (LSI) с рекомендациями по \% вхождения.
- **Commercial & UX Stack:** Необходимые элементы доверия и конверсии.
- **Gap Analysis:** Окно возможностей и точки роста для обхода конкурентов.
